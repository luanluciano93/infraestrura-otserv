# Provisionamento de Hospedagem de Otserv
Este é um manual é usado para provisionar completamente uma máquina Ubuntu para hospedagem OTS.

### Configurações:
- Ubuntu 18.04
- MariaDB e Nginx
- TFS 1.3 e ZnoteACC
